# Golang Flatten

<a name="installation"></a>

## Installation

If you are new on go, read about [how to install go](https://golang.org/doc/install).


## Clone

Clone the repository:

```
$ git clone git@github.com:iver/flatten.git
```

## Execution

Go to golang folder...

```
$ cd flatten/golang/
$ go run flatten.go
```

## Test

If you wish to run the test:

```
$ go test -v .
```
